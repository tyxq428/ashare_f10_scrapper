# W03：前端信息架构与监控优化

## 目标

让普通投研用户只处理业务选择，不需要理解内部并发、报告年份或退避算法。

## 页面结构

### 默认可见

1. 股票代码（默认留空，示例放placeholder）。
2. 研究模式。
3. 官方验证范围（仅在相关模式出现，默认上市以来）。
4. Raw Pack资料范围（仅在相关模式出现）。
5. “开始研究”按钮。

### 高级设置（默认折叠）

- 接口并发数；
- 任务刷新频率；
- 缓存/检查点复用；
- 自动重试轮数和退避秒数；
- Raw Pack文档安全上限。

## 监控优化

- 主任务和阶段状态显示中文；
- 旧任务根据完成组数和Artifact推导状态，不显示`UNKNOWN`；
- 显示官方验证范围、发现文档数、官方事实数、对账数、真实冲突数；
- `COMPLETED_WITH_REVIEW`显示黄色“完成，需复核”；
- 只展示可下载且有明确中文标签的Artifact；
- 隐藏`fact_count`等内部值。

## 验收

- 页面不存在年份输入框；
- 高级参数默认折叠；
- 状态、范围和结果摘要全部可理解；
- 页面在窄屏可用；
- JavaScript语法检查通过。
