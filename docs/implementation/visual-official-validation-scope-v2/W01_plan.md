# W01：基线审计与UX问题清单

## 输入

- 当前`run.html`、`run.js`、`visual_execution.py`、`visual_jobs.py`。
- 已合并的SSE全历史报告生命周期与`run_full_cross_validation`。
- 用户提供的页面截图和实际操作反馈。

## 执行

1. 核对页面所有可见字段是否需要普通投研用户理解。
2. 核对官方验证入口实际调用薄切片还是全历史Runner。
3. 核对旧任务、状态标签和Artifact下载是否存在错误或误导。
4. 核对并行PR文件范围，确定无冲突实施边界。
5. 将全部问题写入集中问题记录。
