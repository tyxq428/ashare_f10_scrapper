# W03结果：前端信息架构与监控优化

## 状态

`COMPLETED`

## 页面优化

默认页面只展示：

1. 股票代码；
2. 研究模式；
3. 官方验证范围；
4. Raw Pack资料范围；
5. 开始研究按钮。

以下项目移入默认折叠的高级设置：

- 接口并发数；
- 页面刷新频率；
- 检查点复用；
- 自动重试轮数与退避；
- Raw Pack文档安全上限。

## 交互修复

- 股票代码默认留空，以placeholder提供示例，防止误运行688521。
- 删除“年度报告年份”和“一季度报告年份”输入。
- 官方验证范围默认“上市以来全部报告（推荐）”。
- 内部英文状态转换为中文业务文案。
- 旧任务根据完成组数和Artifact推导F10、Raw Pack和官方验证状态，不再显示`UNKNOWN`。
- 下载区采用文件Artifact白名单，不再把`fact_count`等内部值渲染为链接。
- 研究冲突、覆盖缺口和部分官方来源不可用均显示“完成，需复核”。
- 官方验证卡片展示范围、报告数、官方事实数、对账数与冲突数。
- 页面窄屏布局和动态Capabilities加载均完成验证。

## 验收

- 静态HTML不再包含年份字段；
- 动态Capabilities提供四档范围和默认全历史；
- JavaScript语法、页面/API健康检查和定向UI测试通过。
