# Windows 可视化使用入口

正常使用不需要输入抓取参数或拼接命令行。

1. 下载或更新仓库的 `main` 分支。
2. 双击 `scripts/start-web.bat`。
3. 浏览器会打开 `http://127.0.0.1:8000/run.html`。
4. 在页面中选择股票代码、任务预设、并发、自动重试、Raw Pack和官方验证参数，然后点击“开始可视化任务”。

任务预设包括：

- F10基础数据包；
- F10 + Raw Pack；
- F10 + 官方交叉验证；
- 完整投研数据包。

任务运行后，可在同一页面查看进度、心跳、失败子任务、自动重试、可选阶段状态和下载文件。

命令行仅保留给CI、排障和高级开发，不作为普通用户默认入口。
